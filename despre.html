Limbajul HTML

HTML (HyperText Markup LAnguage) este un limbaj de marcare utilizat pentru a marca pagini web

Un exemplu de cod HTML il gasiti in imaginea urmatoare care se afla la adresa https://en.wikipedia.org/wiki/HTML
Imaginea are atributul height de valoare 147.

Tag-urile html cele mai utilizate in realizarea tabelelor sunt:

table defineste un tabel
tr un rand intr-un tabel
td o celula intr-un tabel

Observatie: pentru a pune un contur unui tabel se poate (dar nu se recomanda) atributul border="1".
La validare va da eroarea: The border attribute on the table element is obsolete.

Formularele se pot crea cu ajutorul tag-ului form care poate contine tag-urile: imput, textarea, label, select, option, button.
Pentru mai multe detalii consutati cursul tutorial: HTML form and imputs de la adresa https://www.w3schools.com/html/html_formatting.asp

Pentru a crea liste se pot folosi tagurile:

  1. ul- lista neordonata
  2. ol-lista ordonata
  3. li-item din lista
  4. dl-lista cu descrieri
  5. td- termen dintr-o lista cu descrieri
  6. dd- descrierea dintr-o lista cu descrieri

Pentru a insera un document HTML in altul de poate folosi tag-ul <iframe>.
Astfel, un exemplu de mai jos insereaza continutul de la adresa: https://html.spec.whatwg.org/multipage/.
